# Rust notes
<br><br>

#### Compile 
```
rustc src/main.rs
```
<br>

#### Run compiled code
```
./main
```
<br>

****
#### Cargo
```
cargo build
```
OR
```
cargo b --release
```
<br>

Cleaning:
```
cargo clean
```
<br>

Compile and Run:
```
cargo run
```
<br>

Check compilation issues:
```
cargo check
```
